# Visual Editor Improved

## Command Mode

| Command | Meanning             |
| ------- | -------------------- |
| i       | insert               |
| a       | append               |
| u       | undo last operation  |
| v       | select code          |
| p       | paste                |
| x       | delete one character |
| d       | delete selected code |
| dd      | delete current line  |
